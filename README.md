# نظام حضور الطلاب - Student Attendance System

نظام ويب احترافي لإدارة حضور وغياب الطلاب.

## المميزات

- ✅ تسجيل الدخول للأدمن فقط
- ✅ إضافة طلاب جدد (اسم + كود)
- ✅ تسجيل الحضور بالكود
- ✅ أيام التسجيل: الخميس والجمعة فقط
- ✅ جزئين لكل يوم (تسبحة/محاضرة1 + قداس/محاضرة2)
- ✅ منع تسجيل حضور مزدوج
- ✅ صوت عند تسجيل الحضور
- ✅ تحديد اليوم تلقائيًا
- ✅ تسجيل التاريخ والوقت تلقائيًا
- ✅ تسجيل الغياب تلقائيًا
- ✅ البحث عن الطلاب
- ✅ حساب نسبة الحضور
- ✅ لوحة الإحصائيات
- ✅ تصدير التقارير إلى Word

## طريقة التشغيل

### 1. تثبيت المتطلبات

- Node.js
- MongoDB

### 2. تثبيت المكتبات

```bash
cd student-attendance
npm install
```

### 3. تشغيل MongoDB

```bash
mongod
```

### 4. تشغيل السيرفر

```bash
npm start
```

### 5. فتح الموقع

```
http://localhost:3000
```

## بيانات الدخول

- اسم المستخدم: `admin`
- كلمة المرور: `admin123`
